## 기사알파고

- <h3>개요</h3>
  - 유형별 실시간 정보처리기사 문제 풀이 및 기출 문제 다운로드 사이트 제작
  
- <h3>제작과정</h3>
  - 프로젝트명: <b>기사알파고</b> <br>
  - 수행기간:  2018. 11. 15 ~ 2018. 11. 30 <br>
  - 프로젝트 인원:  3명  <br>
  - 나의 기여도:  70% <br>
  - 담당: 데이터베이스 설계, 다운로드, 실시간 문제풀이 기능 구현  <br>
  
- <h3>개발환경</h3>
  - DB : MySQL(8.0.12) <br>
  - PHP(5.2.0) <br>
  - 프레임 워크 : Atom <br>
  - 서버 : Apache(2.4) <br>
  - HTML, CSS, BootStrap, AJAX, JavaScript <br>
